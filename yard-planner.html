Welcome to Yard & Property Planner!

This tool helps you plan and visualize your property layout with an intuitive drag-and-drop interface.

Quick Start:

  1. Define your property boundary or load a template
  2. Upload a background image to trace over (optional)
  3. Drag objects from the sidebar onto your property
  4. Move, resize, and rotate objects as needed
  5. Add setback lines for building regulations
  6. Use Stats to see property coverage and areas
  7. Try the Sun Simulator to see shade patterns
  8. Save your work or export as PNG

Tips:

  • Pan: Shift + drag or middle-mouse drag
  • Zoom: Mouse wheel
  • Rotate: Drag the rotation handle (snaps to 15°)
  • Add Notes: Right-click any object to add a note
  • View Notes: Hover over an object to see its note
  • Delete: Select object and press Delete/Backspace
  • Duplicate: Select object and press Ctrl+D
  • Undo/Redo: Ctrl+Z / Ctrl+Shift+Z

Your work is automatically saved to your browser. You can also save to a file for backup.
